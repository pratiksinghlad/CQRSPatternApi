# Skills Entry Point

> **This folder is a lightweight tool entry point only.**
> The single source of truth for all skills is `AgentSkills/skills/`.

Read `AgentSkills/skills/INDEX.md` as the entry point.

All agents and tools (Copilot, Codex, Claude, Cursor, Antigravity) load skills from `AgentSkills/skills/`.
Do **not** add skill content here. Edit only in `AgentSkills/skills/<skill-name>/SKILL.md`.
