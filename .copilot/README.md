# Copilot Entry Point

This folder is a lightweight tool entry point only.

`AgentSkills/` is the single source of truth for all skills, agents, and memory used by AI tools in this repository.
Use `AgentSkills/skills/INDEX.md`, `AgentSkills/agents/`, and `AgentSkills/memory/index.md` (load only the domain lesson file(s) relevant to your task).

Do not add or edit canonical content under `.copilot/`.
